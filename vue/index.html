从零开始搭建 vue环境

一、安装node环境

  1. 下载地址为：https://nodejs.org/en/

  2. 查看是否安装成功

    1

    node -v

  3. 初始化 packag.json 文件

    1

    npm init -y

  4. 切换淘宝的镜像仓库：http://npm.taobao.org/

    1

    npm install -g cnpm --registry=https://registry.npm.taobao.org

二、搭建vue项目环境

  1. 全局安装vue-cli

    1

    npm install --global vue-cli

    ![npm install](/Users/ratel/Documents/npm install.png)

  2. 进入项目目录，创建一个基于 webpack 模板的新项目

    1

    vue init webpack 项目名

三、安装 element-ui 项目环境

  1. 安装 element-ui 环境

    1

    npm i element-ui -S

    ![install element-ui](/Users/ratel/Documents/install element-ui.png)

  2. Element-ui 快速上后 官方文档

    https://element.eleme.cn/#/zh-CN/component/quickstart
